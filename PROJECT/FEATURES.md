# 기능 명세 — lab904 영수증 정리 앱

---

## 1. 홈 화면 (screen-home)

- 현재 분기 합계 금액 표시
- 프로젝트별 분기 합계 (뱃지)
- 최근 영수증 5건 표시
- 증빙유형 미입력 경고 배너 (클릭 시 해당 목록으로 이동)
- 빠른 업로드 버튼
- 헤더: 사용자명, 드라이브 동기화 상태 아이콘, 설정

---

## 2. 영수증 등록 (screen-upload)

### 사진 업로드 모드
- 카메라 촬영 / 갤러리 선택
- 이미지 미리보기
- AI 용도 자동 분류 (Claude API)

### 직접 입력 모드
- 금액, 용도(매장명) 입력

### 공통 입력 필드
- 날짜: 오늘 / 어제 / 직접 선택 (달력)
- 프로젝트 선택 (칩)
- 결제수단: 카드 / 현금 / 이체 / 중고거래
- 카드 선택 (카드 결제 시)
- 계정과목 선택 (토글/아코디언)
- 증빙유형 선택
- 중고거래 추가 정보 (플랫폼, 결제자, 정산 여부)

---

## 3. 목록 (screen-list)

- 전체 영수증 목록
- 필터: 프로젝트 / 결제수단 / 날짜범위 / 계정과목
- 정렬: 날짜순 / 금액순
- 검색 (용도, 프로젝트명)
- 각 항목 클릭 → 상세보기

### 상세보기 (시트)
- 영수증 전체 정보
- 이미지 확대/축소
- 이미지 다운로드
- 공유 (카드 이미지, 텍스트, 사업자 이메일)
- 수정 / 삭제

---

## 4. 조회 (screen-viewer)

- 기간 선택 (이번달 / 지난달 / 이번분기 / 직접선택)
- 프로젝트별 그룹핑
- 합계 금액 표시
- 세무 포함/제외 필터
- 카드 제외 토글

---

## 5. 정산 (screen-settle)

- 분기 선택 (Q1~Q4)
- 전체 합계 / 세무포함 합계
- 카드 제외 토글
- 항목별 상세 목록
- 내보내기:
  - 엑셀 (.xlsx) — 전체내역 + 프로젝트별 시트
  - CSV
  - ZIP (엑셀 + 사진)

---

## 6. 설정 (screen-settings)

### 사용자
- 로그인/로그아웃

### 직원 관리 (관리자)
- 직원 추가/삭제
- 관리자 PIN 변경 (기본: 0000)

### 카드 관리
- 카드 추가/수정/삭제
- 카드명, 색상, 마지막 4자리
- 카드별 세무 제외 설정

### 사업자 정보
- 사업자등록번호
- 이메일
- 사업자등록증 (이미지)

### Google Drive 연동
- Apps Script URL 설정
- 수동 동기화 버튼

### 프로젝트 관리
- 프로젝트 추가/수정/삭제
- 완료 처리
